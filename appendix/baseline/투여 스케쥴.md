# GLP-1 주사제 투여 주기 및 일정 관리 가이드라인

## 1. 기본 투여 주기

### 1.1 주 1회 제제 (세마글루타이드, 티르제파타이드)

**표준 투여 원칙:**
- **고정된 요일에 주 1회 투여**
- 예: 매주 월요일 오전
- 시간대는 유연 (아침/점심/저녁 모두 가능)
- 식사와 무관하게 투여 가능

**장점:**
- 정해진 요일로 기억하기 쉬움
- 생활 패턴에 맞춰 선택 가능

---

## 2. 투여 일정 변경 상황별 가이드라인

### 2.1 **상황 1: 정규 투여일을 놓친 경우 (지연 투여)**

#### Case A: 다음 투여일까지 **2일 이상** 남은 경우

**✅ 대처법:**
- **즉시 투여** (생각난 즉시)
- 그 다음부터는 원래 정해진 요일에 투여

**예시:**
```
- 원래 투여일: 매주 월요일
- 놓친 상황: 월요일에 투여 못함
- 화요일에 발견 → 화요일 즉시 투여
- 다음 투여: 다음 주 월요일 (원래 스케줄로 복귀)
```

**주의사항:**
- 가능한 한 빨리 투여할수록 좋음
- 투여 후 다음 정규 투여일까지 최소 2일 간격 확보


#### Case B: 다음 투여일까지 **2일 미만** 남은 경우

**⚠️ 대처법:**
- **건너뛰고 다음 정규 투여일에 투여**
- 2회 용량을 한 번에 투여하지 않음

**예시:**
```
- 원래 투여일: 매주 월요일
- 놓친 상황: 월요일에 투여 못함
- 토요일에 발견 (다음 월요일까지 2일)
- 조치: 토요일은 투여 안 함
- 다음 투여: 월요일 (정규 스케줄)
```

**이유:**
- 최소 **48시간(2일) 간격** 필요
- 간격이 짧으면 부작용 위험 증가


#### Case C: 여러 회 연속으로 놓친 경우

**🚨 대처법:**
- **의료진에게 연락하여 상담**
- 일반적으로 다음 정규 투여일부터 재시작
- 경우에 따라 용량 재조정 필요할 수 있음

**예시:**
```
- 2주 이상 투여 안 함
- 조치: 주치의 상담 후 결정
- 가능한 옵션:
  1) 같은 용량으로 재시작
  2) 한 단계 낮은 용량으로 재시작
  3) 초기 용량부터 다시 증량
```

**고려사항:**
- 중단 기간이 길수록 부작용 재발 위험
- 특히 위장관 부작용 다시 나타날 수 있음

---

### 2.2 **상황 2: 투여일을 앞당겨야 하는 경우 (조기 투여)**

#### Case A: 여행, 일정 등으로 미리 맞아야 하는 경우

**📋 기본 원칙:**
- 정규 투여일 **최대 2일 전**까지 조기 투여 가능
- 2일 이상 앞당기는 것은 권장하지 않음

**허용되는 조기 투여:**
```
✅ 가능한 경우:
- 원래 투여일: 월요일
- 주말 여행으로 금요일 또는 토요일에 투여
- 다음 투여: 다음 주 월요일 (원래 스케줄)

❌ 권장하지 않는 경우:
- 원래 투여일: 월요일
- 수요일이나 목요일에 투여 (3-5일 앞당김)
- 위험: 간격이 너무 짧아 부작용 위험
```

**조기 투여 후 스케줄 조정:**

**옵션 1: 원래 요일 유지**
```
- 원래: 매주 월요일
- 조기 투여: 이번 주 토요일 (2일 앞당김)
- 다음 투여: 다음 주 월요일 (9일 간격)
- 그 다음: 매주 월요일 유지
```

**옵션 2: 새로운 요일로 변경**
```
- 원래: 매주 월요일
- 조기 투여: 이번 주 토요일
- 다음 투여: 다음 주 토요일 (7일 간격)
- 그 다음: 매주 토요일로 변경
```

#### Case B: 영구적으로 투여일을 바꾸고 싶은 경우

**🔄 변경 방법:**

**방법 1: 점진적 변경 (권장)**
```
현재: 매주 월요일
목표: 매주 금요일로 변경

1주차: 월요일 투여
2주차: 수요일 투여 (2일 앞당김)
3주차: 금요일 투여 (2일 앞당김)
4주차 이후: 매주 금요일 유지
```

**방법 2: 한 번에 변경**
- 최소 간격(2일) 원칙만 지키면 가능
- 단, 간격이 너무 길어지면 약효 감소 가능

**예시:**
```
원래: 매주 월요일
목표: 매주 목요일

조치:
- 이번 주 월요일 투여
- 다음 주 목요일 투여 (10일 간격)
- 그 다음부터: 매주 목요일
```

**주의사항:**
- 간격이 **14일(2주)을 초과하면 안 됨**
- 가능하면 5-9일 범위 내로 조정

### 2.3 **상황 3: 특수 상황별 가이드**

#### Case A: 수술/시술이 예정된 경우

**🏥 시술 전 중단 원칙:**

**주 1회 제제 (세마글루타이드, 티르제파타이드):**
- **최소 1주일 전 마지막 투여**
- 반감기가 길어 1주일 전 투여 필요

**예시:**
```
수술일: 12월 16일 (월요일)
원래 투여일: 매주 월요일

조치:
- 12월 2일 (월): 투여 ✅
- 12월 9일 (월): 투여 안 함 ❌ (1주일 전)
- 12월 16일 (월): 수술
- 12월 23일 (월): 투여 재개 (의사 확인 후)
```

**내시경 검사:**
```
검사일: 12월 16일 (월요일)
원래 투여일: 매주 월요일

조치:
- 12월 9일 (월): 투여 안 함 ❌
- 12월 16일 (월): 내시경
- 12월 17일 (화): 투여 재개 가능
```

**응급 수술:**
- 담당 의사/마취과에 GLP-1 약물 투여 사실 반드시 고지
- 마지막 투여 시점 정확히 알려주기
- 위 내용물 확인 위해 위 초음파 검사 가능


#### Case B: 심한 위장관 부작용 발생 시

**🤢 일시적 중단이 필요한 경우:**

**중단 기준:**
- 지속적인 구토로 식사 불가능
- 심한 탈수 증상
- 심한 복통
- 설사로 일상생활 불가

**재시작 원칙:**
```
중단: 부작용으로 2주간 투여 중단
재시작:
- 옵션 1: 같은 용량으로 재시작 (중단 기간 짧으면)
- 옵션 2: 한 단계 낮은 용량 (중단 기간 길면)
- 의사와 상의 필수
```


#### Case C: 약물 변경 시 (예: 세마글루타이드 → 티르제파타이드)

**약물 전환 프로토콜:**

**세마글루타이드 → 티르제파타이드:**
```
마지막 세마글루타이드 투여: 월요일
티르제파타이드 시작: 다음 주 월요일 (7일 후)
용량: 티르제파타이드 2.5mg부터 시작
```

**티르제파타이드 → 세마글루타이드:**
```
마지막 티르제파타이드 투여: 월요일
세마글루타이드 시작: 다음 주 월요일 (7일 후)
용량: 세마글루타이드 0.25mg부터 시작
```

**원칙:**
- 최소 7일 washout 기간
- 새로운 약물은 초기 용량부터 시작
- 이전 약물의 최고 용량과 무관


#### Case D: 용량 증량 주기

**증량 타이밍:**
- 최소 **4주 간격**으로 증량
- 부작용 있으면 현재 용량 더 오래 유지

**예시 - 세마글루타이드:**
```
1-4주차: 0.25mg (매주 월요일)
5-8주차: 0.5mg (매주 월요일)
  → 부작용 심하면: 8-12주차도 0.5mg 유지
9주차 이후: 1mg 고려
```

**증량 시 주의사항:**
- 증량 직후 부작용 재발 가능
- 식이 조절 다시 강화
- 증량 후 4주는 면밀히 관찰

## 3. 최소/최대 투여 간격 요약표

### 3.1 주 1회 제제 (세마글루타이드, 티르제파타이드)

| 상황 | 최소 간격 | 권장 간격 | 최대 간격 | 조치 |
|------|----------|----------|----------|------|
| **정상 투여** | - | **7일** | - | 같은 요일 |
| **조기 투여** | **2일** | 5-7일 | 7일 | 최대 2일 전 가능 |
| **지연 투여** | - | 7일 | **14일** | 2주 초과 시 의사 상담 |
| **투여일 변경** | **2일** | 7일 | 10일 | 점진적 변경 권장 |
| **놓친 후 다음** | **2일** | 7일 | - | 2일 미만이면 skip |

### 3.2 절대 원칙

**🚫 절대 금지:**
1. **2회 용량 동시 투여 금지**
2. **48시간 이내 재투여 금지**
3. **임의로 용량 증량 금지**

**⚠️ 의사 상담 필수:**
1. 2주 이상 중단 후 재시작
2. 3일 이상 조기 투여 필요
3. 심한 부작용 발생
4. 수술/시술 예정

## 4. 실전 시나리오별 Q&A

### Q1. 월요일에 맞아야 하는데 화요일에 생각났어요.
**A:** 즉시 투여하세요 (화요일). 다음 주 월요일부터 다시 정상 스케줄로.

### Q2. 월요일에 맞아야 하는데 토요일에 생각났어요.
**A:** 토요일은 skip. 다음 월요일에 정상 투여. (2일 간격 원칙)

### Q3. 월요일 투여인데 금요일 해외여행 가요.
**A:** 
- **옵션 1**: 토요일에 투여 (2일 앞당김) → 다음 월요일 정상
- **옵션 2**: 금요일 아침에 투여 → 다음 금요일로 투여일 변경

### Q4. 일요일 투여인데 다음 주 수요일에 수술이 있어요.
**A:** 이번 주 일요일은 투여하지 마세요. 수술 후 의사 확인 받고 재개.

### Q5. 2주 동안 부작용으로 투여를 못 했어요.
**A:** 의사 상담 필수. 보통 한 단계 낮은 용량으로 재시작.

### Q6. 약 효과가 약해진 것 같아서 간격을 5일로 줄이고 싶어요.
**A:** 절대 안 됩니다. 용량 증량을 의사와 상의하세요.

### Q7. 실수로 이틀 연속 맞았어요.
**A:** 즉시 의사에게 연락. 다음 정규 투여일까지 중단.

### Q8. 원래 월요일인데 앞으로 금요일로 바꾸고 싶어요.
**A:** 
```
1주차: 월요일 투여
2주차: 수요일 투여 (2일 앞당김)
3주차: 금요일 투여 (2일 앞당김)
4주차~: 매주 금요일
```

## 5. 투여 시간대 관련 가이드

### 5.1 하루 중 언제 맞아야 하나요?

**✅ 자유로운 시간 선택:**
- 아침, 점심, 저녁 **언제든 가능**
- 식사와 무관
- 개인 생활 패턴에 맞춰 선택

**권장 시간대:**

**아침 투여 (가장 흔함):**
```
장점:
- 규칙적으로 기억하기 쉬움
- 부작용 모니터링 용이
- 식사 조절 계획 세우기 좋음

추천 대상:
- 규칙적인 아침 루틴 있는 사람
- 낮 시간 활동적인 사람
```

**저녁/취침 전 투여:**
```
장점:
- 오심 등 부작용이 수면 중 발생
- 다음 날 아침 상쾌하게 시작

추천 대상:
- 오심 부작용 심한 사람
- 낮 시간 부작용 불편한 사람
```

### 5.2 투여 시간대 변경

**같은 날 시간만 바꾸기:**
- 자유롭게 변경 가능
- 예: 월요일 아침 → 월요일 저녁

**주의:**
- 요일은 고정하되 시간은 유연하게

## 6. 보관 및 휴대 가이드

### 6.1 여행 시 약물 관리

**국내 여행:**
- 아이스팩과 함께 쿨러백에 보관
- 직사광선 피하기
- 차 안에 장시간 방치 금지

**해외 여행:**
- **냉장 보관 필수** (2-8°C)
- 기내 반입: 의사 처방전 지참
- 예비 펜 추가 휴대 권장
- 여행지 시간대 고려하여 투여일 조정

**장기 여행 (2주 이상):**
```
예시: 3주 여행
- 출발 전: 여행 중 투여 스케줄 미리 계획
- 옵션 1: 출발 전날 투여 → 7일, 14일 후 투여
- 옵션 2: 투여일을 여행 일정에 맞춰 조정
- 현지 냉장고 확보 확인
```

### 6.2 보관 온도 이탈 시

**실온 노출 (25°C 이하):**
- 최대 **21일**까지 안전
- 이후 냉장고 재보관 불가
- 21일 초과 시 폐기

**고온 노출 (25°C 초과):**
- 즉시 폐기 권장
- 약효 저하 가능성

## 7. 기록 및 관리 팁

### 7.1 투여 기록 방법

**스마트폰 알람:**
```
알람 1: 투여일 아침 9시 "오젬픽 투여일"
알람 2: 투여 3일 전 "금요일 여행? 투여일 조정 필요"
알람 3: 다음 진료 1주일 전 "처방 확인"
```

**캘린더 앱:**
- 매주 반복 일정 등록
- 용량, 주사 부위도 메모

**전용 앱:**
- 당뇨/비만 관리 앱 활용
- 체중, 혈당, 부작용도 함께 기록

### 7.2 체크리스트

**투여 전 확인사항:**
□ 정확한 투여일인가?
□ 마지막 투여 후 최소 2일 경과?
□ 약물이 투명한가? (혼탁하면 폐기)
□ 유효기간 확인?

## 8. 응급 상황 대응

### 8.1 과량 투여 시

**증상:**
- 심한 오심/구토
- 저혈당 (당뇨약 병용 시)
- 심한 복통

**조치:**
1. 즉시 의사/약사에게 연락
2. 다음 정규 투여일까지 중단
3. 증상 모니터링
4. 필요시 응급실 방문

### 8.2 긴급 연락이 필요한 상황

**즉시 의사 연락:**
- 2회 연속 투여
- 심한 알레르기 반응
- 지속적 구토로 탈수
- 심한 복통 (췌장염 의심)
- 3주 이상 중단 후 재시작

## 9. 핵심 요약

### 투여 주기 황금률 5가지:

**1. 정해진 요일 지키기**
   - 같은 요일, 주 1회
   - 시간은 유연하게

**2. 최소 48시간 간격**
   - 어떤 경우에도 2일 이내 재투여 금지
   - 안전성 최우선

**3. 2일 전까지 조기 투여 가능**
   - 그 이상 앞당기기는 권장 안 함
   - 의사와 상의

**4. 2주 초과 지연은 상담 필수**
   - 용량 재조정 필요할 수 있음
   - 부작용 재발 대비

**5. 수술 1주일 전 중단**
   - 흡인 위험 예방
   - 마취과에 반드시 고지

## 10. 자주 묻는 질문 (FAQ)

**Q: 시간이 몇 시간 차이 나도 괜찮나요?**
A: 네, 같은 날이면 몇 시간 차이는 문제없습니다.

**Q: 하루 정도 늦게 맞아도 괜찮나요?**
A: 네, 다음 투여일까지 2일 이상 남았다면 즉시 투여하세요.

**Q: 실수로 이틀 전에 맞았어요. 다음 주는 언제 맞나요?**
A: 원래 정해진 날에 맞으세요. 한 번의 조기 투여는 큰 문제없습니다.

**Q: 약이 떨어졌는데 병원 예약이 3주 후예요.**
A: 즉시 병원에 연락하여 빠른 처방 요청하세요. 3주는 너무 깁니다.

**Q: 여행 중 냉장고가 없어요.**
A: 21일 이내면 실온 보관 가능합니다. 직사광선만 피하세요.
