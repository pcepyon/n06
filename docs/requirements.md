# GLP-1 치료 관리 MVP 요구사항 문서 (Phase 0)

## 1. 제품 개요 및 비전

### 1.1. 제품 개요
본 제품은 GLP-1 주사제 사용자들을 위한 디지털 솔루션으로, 복잡한 **개인별 투여 스케줄**을 자동 관리하고, **주요 증상 및 체중 변화 기록**을 데이터 기반의 보고서로 자동 생성하여 의료진과의 커뮤니케이션 효율과 환자의 치료 안전성을 극대화하는 것을 목표로 한다.

### 1.2. 비전
GLP-1 치료 과정에서 발생하는 모든 불확실성(스케줄 혼동, 증상 보고의 비일관성)을 해소하여, 사용자가 안전하고 효율적으로 치료 목표에 도달할 수 있도록 돕는 필수적인 디지털 파트너가 된다.

---

## 2. 타겟 사용자와 핵심 시나리오

### 2.1. 타겟 사용자
GLP-1 주사제를 복용하는 사용자 (자가 투여에 익숙하지만, 스케줄 관리와 데이터 기록에 비효율을 겪는 모든 연령대의 사용자).

### 2.2. 핵심 시나리오 (P0 기능 기반)

| 시나리오 No. | 시나리오 |
| :--- | :--- |
| **SC1. 일상 투여 및 기록** | 사용자는 설정된 투여일과 용량 알림을 받고 주사를 투여한다. 투여 후 메스꺼움이 발생하자, 앱에 메스꺼움 강도를 기록하고, 기록 즉시 앱에서 해당 증상에 대한 **공식 대처 가이드**를 확인하여 안전하게 대처한다. |
| **SC2. 의료진 상담 및 처방** | 사용자는 정기 검진일에 맞춰 앱에 기록된 지난 치료 기간의 **용량 변화, 체중 변화, 부작용 기록**이 통합된 시각화 보고서(PDF)를 3초 이내에 생성한다. 이 보고서를 의료진에게 제출하고 데이터 기반의 피드백을 통해 다음 처방을 확정한다. |

## 3. 기능 목록

| ID | 기능명 | 설명 |
| :--- | :--- | :--- |
| **F001** | **개인화된 투여 스케줄러** | 초기 처방 정보를 기반으로 전체 스케줄을 자동 생성한다. 사용자 또는 의료진 입력에 따라 다음 투여일/용량을 수동으로 변경할 경우, 이후의 모든 스케줄과 알림을 즉시(1초 이내) 자동 재계산하여 반영해야 한다. |
| **F002** | **주요 증상 및 체중 간편 기록** | 투여일 및 매일의 **체중**과 **주요 부작용 (메스꺼움, 구토, 변비 등)**의 심각도를 **5점 척도(VAS)**로 최소한의 터치(3회 이내)로 간편하게 기록할 수 있도록 한다. |
| **F003** | **자동 의료진 보고서 생성** | F001의 용량 변화 시점, F002의 체중 변화 추이 그래프, 부작용 강도/빈도 시각화 그래프를 통합한 **PDF 형식의 보고서**를 3초 이내에 자동 생성하고 내보낼 수 있어야 한다. |
| **F004** | **연계형 부작용 대처 가이드** | F002에서 사용자가 특정 부작용을 기록하거나 심각도를 높게 설정했을 때, 해당 증상에 특화된 **공식적이고 검증된 대처 가이드라인**을 즉시 표시하여 안전성을 확보한다. |
