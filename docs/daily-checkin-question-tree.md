# 데일리 체크인 질문 트리 설계

> 작성일: 2025-12-02
> 원칙: 일상적 안부 → 자연스러운 파생 → 필요시 상세/안내

---

## 전체 구조 개요

```
┌─────────────────────────────────────────────────────────────┐
│                     데일리 체크인                            │
│                   "오늘 하루는 어떠셨어요?"                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  📊 체중 입력 (수치)                                        │
│                                                             │
│  Q1. 🍽️ 식사 질문 ─────→ 메스꺼움/구토 파생                │
│  Q2. 💧 수분 질문 ─────→ 탈수 위험 파생                     │
│  Q3. 😌 속 편안함 질문 ──→ 복통/속쓰림 파생 → 췌장염/담낭염 │
│  Q4. 🚽 화장실 질문 ────→ 변비/설사 파생 → 장폐색          │
│  Q5. ⚡ 에너지 질문 ────→ 피로/어지러움 파생 → 저혈당       │
│  Q6. 😊 기분 질문 ─────→ (선택적)                          │
│                                                             │
│  ✅ 완료 + 맞춤 피드백                                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Q0. 체중 입력 (선택)

### UI 구조
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  📊 오늘 체중을 입력해주세요                                │
│                                                             │
│  ┌─────────────────────────────────────────┐               │
│  │                                         │               │
│  │          [ 85.2 ] kg                    │               │
│  │                                         │               │
│  │  어제: 85.5kg                           │               │
│  │                                         │               │
│  └─────────────────────────────────────────┘               │
│                                                             │
│           ┌────────────────────────┐                       │
│           │         다음           │                       │
│           └────────────────────────┘                       │
│                                                             │
│            건너뛰기 (나중에 기록해도 괜찮아요)               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 입력 사양
- **입력 타입**: 소수점 키보드 (decimal)
- **단위**: kg (고정 표시)
- **범위**: 30.0 ~ 300.0 kg
- **소수점**: 최대 1자리
- **이전 값**: 가장 최근 체중 표시 (없으면 숨김)

### 분기 로직
```
[다음 버튼 클릭]
  → 입력값 유효성 검증
  → weight_logs 테이블에 저장
  → Q1 식사 질문으로 이동

[건너뛰기 클릭]
  → 체중 저장 없음
  → context.weightSkipped = true
  → Q1 식사 질문으로 이동
```

### 피드백 (체중 입력 완료 시)
```dart
if (previousWeight != null) {
  final diff = currentWeight - previousWeight;
  if (diff < 0) {
    feedback = "조금 줄었네요! 💚";
  } else if (diff == 0) {
    feedback = "유지하고 계시네요";
  } else {
    feedback = "괜찮아요, 하루하루 변화가 있을 수 있어요";
  }
}
```

## Q1. 식사 질문 (식욕/포만감 + 메스꺼움)

### 메인 질문
```
🍽️ 오늘 식사는 어떠셨어요?

┌──────────┐  ┌──────────┐  ┌──────────┐
│ 😋 잘    │  │ 😐 적당히 │  │ 😣 좀    │
│ 먹었어요 │  │ 먹었어요  │  │ 힘들었어요│
└──────────┘  └──────────┘  └──────────┘
```

### 분기 로직

```
[잘 먹었어요]
  → 임상 데이터: appetite_score = 5 (좋음)
  → 다음 질문으로

[적당히 먹었어요]
  → 임상 데이터: appetite_score = 3 (보통)
  → 다음 질문으로

[좀 힘들었어요]
  → 파생 질문 Q1-1로
```

### Q1-1. 파생: 식사가 힘들었던 이유
```
혹시 어떤 점이 힘드셨어요?

┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 🤢 속이      │  │ 😶 입맛이    │  │ 🍽️ 조금만   │
│ 메스꺼웠어요 │  │ 없었어요     │  │ 먹어도 배불러요│
└──────────────┘  └──────────────┘  └──────────────┘
```

**분기:**
```
[속이 메스꺼웠어요]
  → 임상 데이터: appetite_score = 2, nausea = true
  → 파생 질문 Q1-1a로 (메스꺼움 상세)

[입맛이 없었어요]
  → 임상 데이터: appetite_score = 2, low_appetite = true
  → 💚 "입맛이 없는 건 약이 작용하고 있다는 신호일 수 있어요"
  → 다음 질문으로

[조금만 먹어도 배불러요]
  → 임상 데이터: appetite_score = 2, early_satiety = true
  → 💚 "포만감이 빨리 오는 건 약이 잘 작용하고 있는 거예요"
  → 다음 질문으로
```

### Q1-1a. 메스꺼움 상세
```
메스꺼움이 어느 정도였나요?

┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 😐 살짝      │  │ 😣 식사하기  │  │ 🤮 물도     │
│ 느꼈어요     │  │ 힘들 정도    │  │ 힘들었어요   │
└──────────────┘  └──────────────┘  └──────────────┘
```

**분기:**
```
[살짝 느꼈어요]
  → 임상 데이터: nausea_severity = 1 (mild)
  → 💚 "가벼운 메스꺼움은 흔히 있어요. 며칠 내에 나아질 거예요"
  → 다음 질문으로

[식사하기 힘들 정도]
  → 임상 데이터: nausea_severity = 2 (moderate)
  → 파생 질문 Q1-1b로 (구토 여부)

[물도 힘들었어요]
  → 임상 데이터: nausea_severity = 3 (severe)
  → 파생 질문 Q1-1b로 (구토 여부) + 🚨 탈수 주의 플래그
```

### Q1-1b. 구토 여부
```
혹시 토하신 적도 있으셨나요?

┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 😌 아니요    │  │ 😣 1-2번     │  │ 🤮 여러 번   │
│              │  │ 있었어요     │  │ (3번 이상)   │
└──────────────┘  └──────────────┘  └──────────────┘
```

**분기:**
```
[아니요]
  → 임상 데이터: vomiting = 0
  → 다음 질문으로

[1-2번 있었어요]
  → 임상 데이터: vomiting = 1-2
  → 💚 "힘드셨죠. 물을 조금씩 자주 마셔보세요"
  → 다음 질문으로

[여러 번 (3번 이상)]
  → 임상 데이터: vomiting = 3+
  → 🚨 탈수 위험 체크 (Q2에서 종합 판단)
  → 다음 질문으로
```

### 메인 질문

```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 💧 충분히    │  │ 💧 좀 적게   │  │ 😰 거의 못   │
│ 마셨어요     │  │ 마신 것 같아요│  │ 마셨어요     │
└──────────────┘  └──────────────┘  └──────────────┘
```

### 분기 로직
```
[충분히 마셨어요]
  → 임상 데이터: hydration = good
  → 다음 질문으로

[좀 적게 마신 것 같아요]
  → 임상 데이터: hydration = moderate
  → 💚 "내일은 물을 조금 더 챙겨보세요"
  → 다음 질문으로

[거의 못 마셨어요]
  → 파생 질문 Q2-1로
```

### Q2-1. 파생: 수분 섭취 어려움
```
물 마시기가 힘드셨나요?

┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 😶 그냥      │  │ 🤢 마시면    │  │ 🤮 마셔도    │
│ 깜빡했어요   │  │ 속이 안좋아서 │  │ 다 토해요    │
└──────────────┘  └──────────────┘  └──────────────┘
```

**분기:**
```
[그냥 깜빡했어요]
  → 임상 데이터: hydration = poor, reason = forgot
  → 💚 "내일은 알람을 맞춰보는 건 어떨까요?"
  → 다음 질문으로

[마시면 속이 안좋아서]
  → 임상 데이터: hydration = poor, reason = nausea
  → 💚 "조금씩 자주 마셔보세요. 이온음료도 좋아요"
  → 다음 질문으로

[마셔도 다 토해요]
  → 임상 데이터: hydration = severe_risk, cannot_keep_fluids = true
  → 🚨🚨 Red Flag: 심한 탈수 위험
  → Q1 구토 데이터와 종합 판단
```

### 🚨 탈수 Red Flag 종합 판단
```dart
if (cannot_keep_fluids == true ||
    (vomiting >= 3 && hydration == poor)) {

  // Red Flag 안내 표시
  showGuidance(
    level: moderate,
    message: """
      💛 수분 섭취가 어려우시군요.

      지금 가장 중요한 건 조금이라도 수분을 유지하는 거예요.
      • 이온음료를 한 모금씩 자주 마셔보세요
      • 얼음을 입에 물고 있는 것도 도움이 돼요

      만약 계속 물을 못 드시겠으면,
      오늘 중으로 병원에 들러보시는 게 좋겠어요.
    """
  );
}
```

### 메인 질문
```
😌 속은 편하셨어요?

┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 😊 네,       │  │ 😐 좀        │  │ 😣 많이      │
│ 괜찮았어요   │  │ 불편했어요    │  │ 불편했어요   │
└──────────────┘  └──────────────┘  └──────────────┘
```

### 분기 로직
```
[네, 괜찮았어요]
  → 임상 데이터: gi_comfort = good
  → 다음 질문으로

[좀 불편했어요]
  → 파생 질문 Q3-1로

[많이 불편했어요]
  → 파생 질문 Q3-1로 + 🚨 주의 깊게 확인
```

### Q3-1. 파생: 불편함의 종류
```
어떤 불편함이 있으셨어요?

┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│ 🔥 속이      │  │ 😣 배가      │  │ 🫃 배가      │
│ 쓰렸어요     │  │ 아팠어요     │  │ 빵빵했어요   │
└──────────────┘  └──────────────┘  └──────────────┘
```

**분기:**
```
[속이 쓰렸어요]
  → 임상 데이터: heartburn = true
  → 💚 "식후 바로 눕지 않는 게 도움이 돼요"
  → 다음 질문으로

[배가 아팠어요]
  → 파생 질문 Q3-2로 (복통 위치) ⚠️ 중요

[배가 빵빵했어요]
  → 임상 데이터: bloating = true
  → 파생 질문 Q4로 연결 (화장실 질문)
```

### Q3-2. 복통 위치 (⚠️ 췌장염/담낭염 감지 핵심)
```
어디가 아프셨어요?

        ┌─────────────────────────────┐
        │    ┌───┐         ┌───┐     │
        │    │ A │ 명치    │ B │     │
        │    │   │ 윗배    │   │오른쪽│
        │    └───┘         └───┘윗배  │
        │         ┌───┐              │
        │         │ C │              │
        │         │   │ 배꼽 주변    │
        │         └───┘              │
        │    ┌───┐         ┌───┐     │
        │    │ D │         │ E │     │
        │    │   │         │   │     │
        │    └───┘ 아랫배  └───┘     │
        └─────────────────────────────┘

[A: 명치/윗배] [B: 오른쪽 윗배] [C: 배꼽 주변] [D/E: 아랫배]
```

**분기:**
```
[A: 명치/윗배] 또는 [C: 배꼽 주변]
  → 임상 데이터: pain_location = upper_abdomen / periumbilical
  → 파생 질문 Q3-3으로 (심각도 + 등 방사) ⚠️ 췌장염 체크

[B: 오른쪽 윗배]
  → 임상 데이터: pain_location = right_upper_quadrant
  → 파생 질문 Q3-4로 (심각도 + 발열) ⚠️ 담낭염 체크

[D/E: 아랫배]
  → 임상 데이터: pain_location = lower_abdomen
  → 💚 "아랫배 불편함은 장이 적응하는 과정일 수 있어요"
  → 다음 질문으로
```

### 🚨 췌장염 Red Flag 판단
```dart
if (pain_location in [upper_abdomen, periumbilical] &&
    pain_severity >= moderate &&
    radiation_to_back >= slight &&
    duration >= hours) {

  // Red Flag 안내
  showGuidance(
    level: high,
    message: """
      💛 오늘 기록해주신 증상이 조금 확인이 필요해 보여요.

      윗배 통증이 등 쪽으로도 느껴지고,
      몇 시간 이상 지속되셨군요.

      이런 경우 드물지만 확인이 필요할 때가 있어요.
      오늘 중으로 가까운 병원에 들러서
      한 번 확인받아 보시는 게 안심이 될 것 같아요.

      💡 응급실이 아니어도 괜찮아요.
         가까운 내과에서 확인받으시면 돼요.
    """
  );
}
```

## Q4. 화장실 질문 (배변 + 변비/설사 → 장폐색)

## 완료 + 맞춤 피드백

### 피드백 생성 로직

```dart
String generateFeedback(CheckinData data) {
  // 1. 긍정적 요소 찾기
  List<String> positives = [];

  if (data.appetite >= good) {
    positives.add("식사를 잘 하셨네요");
  }
  if (data.hydration == good) {
    positives.add("수분 섭취도 충분히 하셨고요");
  }
  if (data.energy >= normal) {
    positives.add("에너지도 괜찮으셨군요");
  }
  if (data.noSymptoms) {
    positives.add("오늘 특별한 불편함 없이 잘 보내셨네요");
  }

  // 2. 격려 요소
  List<String> encouragements = [];

  if (data.hadSymptoms) {
    encouragements.add("몸이 적응하는 중이에요. 잘 견디고 계세요");
  }
  if (data.consecutiveDays >= 3) {
    encouragements.add("벌써 ${data.consecutiveDays}일째 기록 중이시네요!");
  }

  // 3. 메시지 조합
  return buildMessage(positives, encouragements);
}
```
