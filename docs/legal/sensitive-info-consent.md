## 1. 민감정보의 수집·이용 목적

| 목적 | 설명 |
|------|------|
| **투여 스케줄 관리** | GLP-1 약물 투여 일정 설정, 알림 제공, 투여 기록 관리 |
| **건강 상태 추적** | 체중 변화 기록 및 분석, 목표 달성 현황 제공 |
| **증상 모니터링** | 일일 컨디션 체크, 부작용 증상 기록 및 대처 가이드 제공 |
| **개인화 인사이트** | 기록 데이터 기반 진행 현황 시각화 및 맞춤 정보 제공 |
| **의료진 상담 지원** | 데이터 공유 모드를 통한 의료진 상담 시 기록 열람 지원 |

## 4. 동의 거부권 및 불이익 안내

이용자는 민감정보 수집·이용에 대한 동의를 거부할 권리가 있습니다.

**단, 동의를 거부하실 경우 다음과 같은 서비스 이용에 제한이 있습니다:**

| 동의 거부 시 이용 불가 서비스 |
|------------------------------|
| 투여 스케줄 관리 및 알림 서비스 |
| 체중 기록 및 변화 추이 분석 |
| 증상 기록 및 대처 가이드 |
| 진행 현황 대시보드 |
| 의료진 상담용 데이터 공유 |

민감정보 수집·이용 동의는 서비스의 핵심 기능 제공을 위해 필수적이며, **동의하지 않으실 경우 비우당 서비스를 이용하실 수 없습니다.**

## 5. 민감정보의 안전성 확보 조치

개발자는 민감정보의 안전한 처리를 위해 다음과 같은 조치를 취하고 있습니다:

- **암호화**: 모든 건강정보는 암호화되어 저장 및 전송됩니다 (TLS 1.2 이상)
- **접근 제어**: Row Level Security(RLS)를 통해 본인의 데이터만 접근 가능
- **접속 기록**: 개인정보처리시스템 접속 기록 2년 이상 보관
- **데이터 분리**: 민감정보는 일반 개인정보와 분리하여 관리
