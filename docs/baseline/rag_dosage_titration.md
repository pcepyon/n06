# GLP-1 수용체 작용제 용량 및 적정 스케줄 가이드

## 문서 개요
본 문서는 Wegovy(세마글루타이드)와 Mounjaro(티르제파타이드)의 FDA/EMA/KFDA 승인 용량 및 적정 스케줄에 대한 공식 지침을 제공합니다.

**최종 업데이트**: 2025년 11월
**출처**: FDA, EMA, KFDA 공식 승인 문서 및 2025년 임상시험 데이터

---

## 1. Wegovy (세마글루타이드) 용량 프로토콜

### 1.1 표준 용량 적정 (FDA/EMA 승인)

**투여 빈도**: 주 1회 피하주사

**적정 스케줄** (총 16주):
- 1-4주차: 0.25mg
- 5-8주차: 0.5mg  
- 9-12주차: 1mg
- 13-16주차: 1.7mg
- 17주차 이후: 2.4mg (유지용량)

**용량 증량 원칙**:
- 각 용량 단계는 최소 4주 유지
- 환자가 내약하지 못할 경우 4주 추가 지연 가능
- 중단하지 말고 용량 증량을 지연시킬 것

### 1.2 고용량 프로토콜 (2025년 신규)

**7.2mg 용량** (STEP UP Trial, 2025):
- 2.4mg 유지 후 추가 적정 가능
- 평균 체중 감량: 20.7% (72주차)
- 25% 이상 감량 달성률: 33%
- 2.4mg 대비 우수한 효능 입증

**적용 대상**:
- 2.4mg에서 체중 감량 목표 미달성
- 2.4mg 용량에서 우수한 내약성
- 추가 체중 감량 필요 환자

### 1.3 누락 용량 관리

**5일 이내 누락**:
- 발견 즉시 투여
- 다음 주차 정상 스케줄 재개

**5일 이상 누락**:
- 누락된 용량 건너뛰기
- 다음 예정 투여일까지 대기
- 다시 적정을 시작할 필요 없음 (유지용량 도달 후)

### 1.4 주사 기술

**투여 부위**:
- 복부 (배꼽 주변 5cm 제외)
- 허벅지 전면
- 상완 후면

**부위 순환**:
- 매주 다른 부위 사용 권장
- 같은 부위 재사용 시 최소 1주 간격

**투여 시간**:
- 하루 중 언제든 가능
- 식사와 무관
- 부작용 최소화를 위해 취침 전 권장

## 2. Mounjaro (티르제파타이드) 용량 프로토콜

### 2.1 표준 용량 적정 (FDA 승인)

**투여 빈도**: 주 1회 피하주사

**적정 스케줄**:
- 1-4주차: 2.5mg (시작용량)
- 5주차 이후: 4주마다 2.5mg씩 증량

**용량 단계**:
- 2.5mg → 5mg → 7.5mg → 10mg → 12.5mg → 15mg (최대용량)

**증량 결정**:
- 내약성 및 효능에 따라 개별화
- 환자가 특정 용량에서 만족스러운 결과 시 유지 가능
- 부작용 발생 시 현재 용량에서 4주 추가 유지

### 2.2 용량별 효능 데이터 (SURMOUNT 시리즈)

**72주차 평균 체중 감량**:
- 5mg: 15.0%
- 10mg: 19.5%
- 15mg: 20.9%

**25% 이상 체중 감량 달성률**:
- 5mg: 23%
- 10mg: 36%
- 15mg: 36.2%

### 2.3 실제 임상 적용 패턴

**Veradigm 데이터베이스 분석 (2025)**:
- 환자의 56.2%가 6개월 시점에 <10mg 용량 유지
- 임상시험보다 느린 용량 증량 속도
- 개별 내약성에 따른 용량 조정 필요

## 3. 세마글루타이드 vs 티르제파타이드 직접 비교

### 3.1 SURMOUNT-5 연구 결과 (2025)

**72주차 체중 감량**:
- 티르제파타이드 15mg: 20.2%
- 세마글루타이드 2.4mg: 13.7%
- 차이: 6.5%p (P<0.001)

**25% 이상 감량 달성률**:
- 티르제파타이드: 32%
- 세마글루타이드: 16%

**부작용으로 인한 중단율**:
- 티르제파타이드: 6.1%
- 세마글루타이드: 8.0%

## 4. 한국 식약처(KFDA) 승인 현황

### 4.1 Wegovy (세마글루타이드)
- **승인일**: 2023년
- **시판 개시**: 2024년 10월
- **승인 용량**: 0.25mg - 2.4mg
- **적응증**: 비만 및 과체중 관리

### 4.2 Mounjaro (티르제파타이드)
- **당뇨 승인**: 2023년

- **수면무호흡증 승인**: 2025년 9월
- **승인 용량**: 2.5mg - 15mg

## 5. 용량 조정 시 고려사항

### 5.1 용량 증량 지연이 필요한 경우
- 중등도 이상의 위장관 부작용
- 오심/구토로 인한 수분 섭취 곤란
- 환자의 요청
- 급격한 체중 감량 (주당 >1kg)

### 5.2 용량 유지가 적절한 경우
- 만족스러운 체중 감량 달성
- 현재 용량에서 우수한 내약성
- 부작용과 효능의 균형
- 환자의 삶의 질 향상

### 5.3 용량 감량이 필요한 경우
- 지속적인 심한 부작용
- 의학적 합병증 발생
- 환자가 용량 증량을 견디지 못함
- 일시적 감량 후 재증량 시도 가능

## 6. 특수 상황 용량 조정

### 6.1 신장애 환자
- 경증-중등도: 용량 조정 불필요
- 중증: 주의 깊은 모니터링
- 투석: 사용 경험 제한적

### 6.2 간장애 환자
- 경증-중등도: 용량 조정 불필요
- 중증: 임상 데이터 부족, 주의

### 6.3 고령 환자 (≥65세)
- 용량 조정 불필요
- 단, 신기능 저하 고려
- 더 천천히 증량 고려

### 6.4 수술 전 환자
- 수술 7일 이상 전 투여 중단
- 위 내용물 저류 위험
- 마취과 의사에게 사용 알림 필수

## 7. 임상적 의사결정 알고리즘

### 7.1 초기 용량 선택
```
환자 평가 → 세마글루타이드 0.25mg 또는 티르제파타이드 2.5mg 시작
↓
4주마다 평가:
- 내약성 양호? → 증량
- 부작용 있음? → 현재 용량 유지 4주
- 심한 부작용? → 의사 상담
```

### 7.2 유지용량 결정
```
체중 감량 목표 달성?
↓
예 → 현재 용량 유지
↓
아니오 → 내약성 확인 후 증량 고려
```

## 8. 환자 교육 핵심 메시지

### 8.1 용량 증량에 대한 이해
- 용량 증량은 부작용 최소화 및 효능 최대화를 위함
- 서두르지 말고 신체가 적응할 시간 필요
- 각자의 속도로 진행 가능

### 8.2 일관성의 중요성
- 매주 같은 요일 투여 권장
- 요일 변경 시 간격 준수 (최소 2일, 최대 14일)
- 달력이나 앱으로 추적

### 8.3 용량 오류 예방
- 처방된 펜 농도 확인
- mg과 mL 구별
- 주사 후 펜 보관 방법
